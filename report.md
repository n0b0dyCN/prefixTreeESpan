# PrefixTreeESpan算法实现

此代码实现了python版的PrefixTreeESpan算法，完成嵌入子树挖掘工作。

## 实验数据
本作业在四个数据集上对min_support为0.1，0.05，0.01分别进行了测试。实验机环境为core-i7 3520M + 16G内存，系统环境为Windows 10 WSL。具体实验结果在out文件夹下。

运行时间如下表（单位：s）：

|       | 0.1     | 0.05    | 0.01    |
| ----- | ------- | ------- | ------- |
| CSlog | 5.1476  | 6.3953  | 23.1296 |
| D10   | 2.9976  | 3.5114  | 8.7379  |
| F5    | 4.1968  | 5.9745  | 11.2152 |
| T1M   | 28.6773 | 31.1768 | 71.4527 |

每个支持度挖掘的频繁子树个数如下表：

|       | 0.1 | 0.05 | 0.01 |
| ----- | --- | ---- | ---- |
| CSlog | 2   | 6    | 133  |
| D10   | 6   | 11   | 112  |
| F5    | 16  | 31   | 259  |
| T1M   | 6   | 8    | 73   |
